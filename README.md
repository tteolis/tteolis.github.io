# Trevor Teolis — Academic Website

This repository powers **https://tteolis.github.io** via GitHub Pages.

## Quick start
1. Ensure the repository is named `trevorteolis.github.io` and public.
2. Go to **Settings → Pages** and set Source = Deploy from a branch, Branch = `main`, Folder = `/ (root)`.
3. Edit `index.md`, `research.md`, `publications.md`, `talks.md` directly in GitHub to update content.
4. Put your PDF CV at `assets/cv.pdf` to enable the CV link.

## Notes
- Theme: `jekyll-theme-minimal` (built-in on GitHub Pages).
- MathJax is enabled via kramdown config.
